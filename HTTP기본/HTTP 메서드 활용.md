## HTTP 메서드 활용
### 클라이언트에서 서버로 데이터 전송

- 데이터 전달 방식 크게 2가지
> - 쿼리 파라미터를 통한 데이터 전송 : GET
> - 메시지 바디를 통한 데이터 전송 : POST, PUT, PATCH

- 4가지 경우
> - 정적 데이터 조회 (GET)
> - 동적 데이터 조회 (GET)
> - HTML Form을 통한 데이터 전송 (POST, GET)
> `- 참고 : HTML Form 전송은 GET, POST만 지원`
> - HTTP API를 통한 데이터 전송 (GET, POST, PUT, PATCH)
